# Remove Background

Веб-приложение для автоматического удаления фона с изображений на основе AI-модели U2Net.

## Возможности

- 🤖 Автоматическое удаление фона с помощью нейросети (U2NetP)
- 🖥️ Простой и удобный веб-интерфейс
- 📥 Скачивание результата в формате PNG

## Требования

- Python 3.12+

## Установка

1. Клонируйте репозиторий или перейдите в папку проекта:
```bash
cd /home/yulyak/result_projects/RemoveBackground-main
```

2. Создайте виртуальное окружение:
```bash
python3 -m venv venv
```

3. Активируйте виртуальное окружение:
```bash
source venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

1. Активируйте виртуальное окружение (если ещё не активировано):
```bash
source venv/bin/activate
```

2. Запустите сервер:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost:5000
```

## Использование

1. Перетащите изображение в область загрузки или кликните для выбора файла
2. Дождитесь обработки
3. Скачайте результат в формате PNG

## Структура проекта

| Файл | Описание |
|------|----------|
| `app.py` | Flask-сервер с логикой удаления фона |
| `index.html` | Веб-интерфейс |
| `style.css` | Стили страницы |
| `script.js` | Клиентская логика загрузки и обработки |
| `requirements.txt` | Python-зависимости |

## Зависимости

- **Flask** — веб-фреймворк
- **rembg[cpu]** — удаление фона на базе U2Net (CPU-версия)
- **opencv-python** — обработка масок и изображений
- **Pillow** — работа с изображениями
- **numpy** — матричные вычисления
- **gunicorn** — WSGI-сервер (для продакшена)

## Запуск в продакшене

Для продакшен-развёртывания рекомендуется использовать gunicorn:

```bash
gunicorn -w 2 -b 0.0.0.0:5000 app:app
```
